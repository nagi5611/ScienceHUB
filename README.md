### 利用の概要

**基本方針:** **商用利用以外のあらゆる研究目的**での利用（改変・解析を含む）を許可しています。利用者の所属・法人格は問いません。

| 対象 | 利用可否 |
|------|----------|
| 個人・任意の組織（非商用の研究目的） | ✅ 利用可（改変・解析を含む） |
| 大学・研究機関 | ✅ 自由に利用可（改変・解析を含む） |
| 企業（非商用の研究目的） | ✅ 改変・解析を含めて利用可 |
| 商用利用・サービス化（全利用者） | ❌ 事前の書面許諾が必要 |

- **改変・解析**は、非商用の研究目的の範囲であれば、個人・大学・研究機関・企業を問わず許可しています。
- **商用利用および無断のサービス化**は禁止しています。不明点がある場合、商用目的での利用、個別許諾が必要な場合は下記までご連絡ください。

ScienceHUB

プライバシーポリシー

最終更新日: 2026年7月15日

ScienceHUB（以下「本サービス」）は、MMH Virtual が運営する科学研究向けのファイル共有・プロジェクト管理・スケジュール管理プラットフォームです。 本ポリシーは、本サービス（https://s.mmh-virtual.jp/）における個人情報等の取り扱いについて定めます。

1. 運営者

  • サービス名: ScienceHUB
  • 運営: MMH Virtual
  • お問い合わせ: harumacci94@gmail.com

2. 取得する情報

本サービスでは、次の情報を取得・保存する場合があります。

  • アカウント情報: ユーザー名、表示名、メールアドレス、パスワード（ハッシュ化して保存）、プロフィール画像、ロール（権限）
  • OAuth 連携情報: Google または Microsoft アカウント連携時の識別子、メールアドレス、表示名（ログイン認証に必要な範囲）
  • 利用データ: グループ所属、プロジェクト・タスク、スケジュール予定、お知らせ、3D 印刷予約など本サービスの機能利用に伴うデータ
  • ファイル: クラウドストレージ等にアップロードされたファイルのメタデータおよび内容
  • 技術情報: セッション Cookie、IP アドレス、ブラウザ種別、アクセス日時などのログ（障害対応・セキュリティのため）

3. 利用目的

  • ユーザー認証、アカウント管理、権限に基づく機能の提供
  • ファイル共有、プロジェクト管理、スケジュール管理など本サービス各機能の提供
  • Google カレンダーとの予定同期（管理者が設定した組織用カレンダーへの反映）
  • お知らせ・通知（設定に応じた Discord 連携等を含む）
  • 不正利用の防止、セキュリティ維持、障害対応
  • サービス改善のための統計・分析（個人を特定しない形での利用を原則とします）

4. Google ユーザー情報の取り扱い

本サービスは Google API から受け取った情報を、次の目的にのみ使用します。

  • ログイン（openid / email / profile）: アカウントの作成・認証、プロフィール表示
  • Google カレンダー連携（calendar スコープ）: ScienceHUB 上の予定を、管理者が指定した Google カレンダーに作成・更新・削除するため

Google API から受け取った情報の利用は、 Google API Services User Data Policy （Limited Use 要件を含む）に従います。 当該情報を第三者への販売、広告配信、信用調査等の目的には使用しません。

5. 第三者への提供・委託

本サービスは次の外部サービスを利用します。必要な範囲でデータが処理される場合があります。

  • Cloudflare: ホスティング、データベース（D1）、ファイルストレージ（R2）、CDN
  • Google: OAuth ログイン、Google カレンダー API
  • Microsoft: OAuth ログイン（利用時）
  • Discord: 3D 印刷予約等の通知 Webhook（管理者設定時）

法令に基づく場合を除き、本人の同意なく個人情報を第三者に提供することはありません。

6. 保存期間

データは、本サービスの提供に必要な期間保存します。 アカウント削除または管理者による削除後は、合理的な期間内に削除または匿名化します。 バックアップ等に短期間残存する場合があります。

7. セキュリティ

パスワードのハッシュ化、HTTPS 通信、セッション管理、アクセス権限の制御など、 合理的な安全管理措置を講じます。

8. ユーザーの権利

プロフィール画面から表示名等の変更が可能です。 アカウント情報の開示・訂正・削除を希望する場合は、上記お問い合わせ先までご連絡ください。

9. Cookie

本サービスはログイン状態の維持のため Cookie を使用します。 ブラウザ設定で Cookie を無効にすると、一部機能が利用できなくなる場合があります。

10. 未成年者

本サービスは学校・研究活動向けに提供されています。 未成年者が利用する場合は、保護者または指導者の同意のもとでご利用ください。

11. ポリシーの変更

本ポリシーは必要に応じて改定します。 重要な変更がある場合は、本ページの更新日を変更するなどしてお知らせします。
